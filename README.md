## How to run locally

1. Ensure you have Node.js (>=16) and npm installed.
2. Unzip project and open terminal in project folder.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start dev server:
   ```bash
   npm run dev
   ```
5. Open browser at the URL printed by Vite (usually http://localhost:5173).
